# 📜 李清照 · 金陵《金石录》

> 李清照与金陵金石录 · Li Qingzhao and *Jinshi Lu* in Jinling

一个开放的数字人文项目，聚焦于宋代女词人 **李清照** 的生平及其與金陵（今南京）和《**金石录**》的不解之缘。

## ✦ 关于

李清照（1084–1155），号易安居士，宋代婉约派词人代表，中国古代最杰出的女文学家之一。她与丈夫赵明诚共同编纂的《金石录》，是中国金石学史上里程碑式的著作。

建炎南渡后，李清照辗转流寓金陵。金陵不仅是她人生转折的重要坐标，更在她的词作与《金石录后序》中留下了深刻的印记。
